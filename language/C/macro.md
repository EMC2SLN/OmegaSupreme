### 题1：宏的展开
设有以下宏定义:
``` C
#define N 3+1
#define Y(n) ((N+1)*n)
```

则执行语句z=2*(N+Y(5+1))后，z的值为

```
  A. 60
  B. 190
  C. 248
  D. 上述答案都不对
```

答案：A
``` C
z =2 * (N+Y(5+1))
  =2 * (N+((N+1)*5+1))  //注意由于#define Y(n) ( (N+1)*n) 中n没有用括号括上，所以这里是*5+1，不是*(5+1)
  =2 * (3+1+((3+1+1)*5+1))
  =2 * (4+(5*5+1))
  =2 * (4+26)
  =2 * 30
  =60
```
